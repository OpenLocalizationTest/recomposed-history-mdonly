# Integration accounts

## What is an integration account?
An integration account is an Azure account that allows Enterprise Integration apps to manage artifacts including shemas, maps, certificates, partners and agreements. Any integration app you create will need to use an integration account in order to access a schema, map or certificate, for example.

## What is an integration account used for?

## Create an integration account 

1. On the Azure Portal's home page, select Web + Mobile
2. Select **New** 
3. In the search box at the top, enter **Integration**
4. Select **Enter** on your keyboard
5. Select **Integration Account (preview)** from the search results. This opens the **Integration Account (preview)** blade.
6. Select the **Create** button. This opens the **Integration Account preview** blade. Here you will provide the information that's required to create your integration account.
7. Provide a **Name** for your integration account.
8. Select the **Subscription** you'd like to use
9. Select the **Create new** radio button to create a new resource group. Optionally, you could select an existing resource group if you have an existing on you'd like to use.
10. Select a **Location** and a **Pricing tier**.
11. Select the **Create** button.
