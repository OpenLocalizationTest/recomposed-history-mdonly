# Use sample experiments to create new experiments
When you want to create an experiment, look for examples in Machine Learning team sample experiments and experiments shared by the Machine Learning community.
## Browse the Gallery
1. Go to http://gallery.azureml.net/
2. Browse either by going to **Trending Experiments** or **Microsoft samples**.
3. Use search and filters to narrow results:
![](./media/machine-learning-sample-experiments/image1.jpg) 
4. Click an experiment to learn more about it. To run and/or modify the experiment, click **Open in Studio**.
![](./media/machine-learning-sample-experiments/image2.jpg) 
5.  Sign in with your Microsoft account credentials. If you don’t have a Machine Learning workspace yet, a free trial workspace is created. [Learn what’s included in the Machine Learning free trial](http://azure.microsoft.com/en-us/pricing/details/machine-learning/)
![](./media/machine-learning-sample-experiments/image3.jpg)  

## Use as template in Studio
1. Sign in with your Microsoft account credentials to https://studio.azureml.net, and then click **New*** to create a new experiment 
2. Browse through the Sample content, and pick one to use as a template.
