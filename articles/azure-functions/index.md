# Azure Functions Documentation

Azure Functions is a solution for easily running small pieces of code, or "functions," in the cloud. Azure Functions is a solution to process events with a serverless code architecture.  Azure Functions supports multiple languages and allows you to write only the code you need without worrying about application or infrastructure to run it.
