# Create a new Dropbox API from Azure Marketplace

1. Select Dropbox from Azure Marketplace

2. Select *Settings - Configure required settings*
3. Enter the Dropbox Key into *Account Key*
	- If you don't have a Dropbox Key, follow these steps:
		- Step 1: Sign in new Dropbox [Account here](https://www.dropbox.com/login), Create your Dropbox [APP HERE](https://www.dropbox.com/developers/apps/create)
		- Step 2: There are 2 options available creating a app, Drops-in App & Dropbox API App. You have to select **Dropbox API app** and give the name of your app. 
		- Step 3: Go to [App Console](https://www.dropbox.com/developers/apps) and select your created app. Under settings tab you can see the APP_KEY(App key) and SECRET_KEY(App secret)

4. Click *OK* on *Configure API* blade
5. Click *OK* on *Create API* blade
