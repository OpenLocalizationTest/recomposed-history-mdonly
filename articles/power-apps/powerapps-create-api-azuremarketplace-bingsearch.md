# Create a new Bing Search API from Azure Marketplace

1. Select Bing Search from Azure Marketplace

2. Select *Settings - Configure required settings*
3. Enter the Bing Search Key into *Account Key*
	- If you don't have a Bing Search Key, create a free [Bing Search API](https://datamarket.azure.com/dataset/bing/search) to obtain one
 
4. Click *OK* on *Configure API* blade
5. Click *OK* on *Create API* blade
