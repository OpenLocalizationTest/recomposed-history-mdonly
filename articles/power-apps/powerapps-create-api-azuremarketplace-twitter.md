# Create a new Twitter API from Azure Marketplace

1. Select Twitter from Azure Marketplace

2. Select *Settings - Configure required settings*
3. Enter *App Key* and *App Secret* for Twitter
	- If you don't have a Twitter app key, follow these steps:
		- Create a free application for Twitter at [http://apps.twitter.com](http://apps.twitter.com).
		- When registering for the app, you can put any URL for the website, and leave the callback URL blank

4. Click *OK* on *Configure API* blade
5. Click *OK* on *Create API* blade
