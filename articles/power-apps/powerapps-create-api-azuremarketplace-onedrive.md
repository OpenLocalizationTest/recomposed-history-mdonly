# Create a new OneDrive API from Azure Marketplace

1. Select OneDrive from Azure Marketplace

2. Select *Settings - Configure required settings*
3. Enter the OneDrive Key into *Account Key*
	- If you don't have a OneDrive Key, follow these [instructions](https://dev.onedrive.com/app-registration.htm#register-your-app-for-onedrive) to obtain one.

4. Click *OK* on *Configure API* blade
5. Click *OK* on *Create API* blade
