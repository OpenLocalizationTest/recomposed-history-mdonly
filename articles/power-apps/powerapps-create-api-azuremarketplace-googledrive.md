# Create a new Google Drive API from Azure Marketplace

1. Select Google Drive from Azure Marketplace

2. Select *Settings - Configure required settings*
3. Enter *App Key* and *App Secret* for Google Drive
	- If you don't have a Google Drive Key, go [here](https://console.developers.google.com/) to obtain one.

4. Click *OK* on *Configure API* blade
5. Click *OK* on *Create API* blade
