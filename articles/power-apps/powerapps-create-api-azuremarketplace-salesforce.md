# Create a new Salesforce API from Azure Marketplace

1. Select Salesforce from Azure Marketplace

2. Select *Settings - Configure required settings*
3. Enter *App Key* and *App Secret* for Salesforce
	- If you don't have a Salesforce key, follow [these instructions](http://feedback.uservoice.com/knowledgebase/articles/235661-get-your-key-and-secret-from-salesforce).

4. Click *OK* on *Configure API* blade
5. Click *OK* on *Create API* blade
