#Data Lake Analytics Documentation

Data Lake Analytics is a distributed systems platform that makes it easy to package, deploy, and manage scalable and reliable microservices and addresses the significant challenges in developing and managing cloud applications.  Data Lake Analytics represents the next-generation middleware platform for building and managing these enterprise-class, Tier-1 cloud-scale applications.
