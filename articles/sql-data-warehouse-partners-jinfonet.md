# Jinfonet
![][logo]

## Who we are
[Jinfonet Software][website] provides reporting and data visualization solutions for the enterprise. Their flagship product JReport is an embeddable BI platform that enables users to create reports, dashboards, and data analysis on cloud, big data, and transactional databases. Jinfonet's intuitive approach to design, deployment, and interaction simplifies the work for both developers and users.

## Solution introduction
JReport provides an embedded reporting and data analysis solution on Microsoft Azure SQL Data Warehouse. The joint solution provides a high performance, scalable BI platform that integrates into any application. Visualizing data from Azure SQL Data Warehouse empowers users to perform their own reporting and data discovery for agile, on-the-fly decision-making.
