# Azure Active Directory Domain Services preview

## Supported Azure Regions
Azure Active Directory Domain Services are available in the following Azure regions:

- United States
  - Central US
  - East US
  - East US2
  - North Central US
  - South Central US
  - West US

- Europe
  - North Europe
  - West Europe

- Asia
  - East Asia
  - Southeast Asia
