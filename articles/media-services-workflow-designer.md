#Workflow Designer

##Overview
The **Workflow Designer** is a standalone tool that is used to create workflows/graphs for encoding with **Media Encoder Premium Workflow**.

You can also use this tool to [modify existing workflows](#existing_workflows). 

>[AZURE.NOTE]To get a copy of the Workflow Designer tool, please contact mepd@microsoft.com.


Once you create the workflow file, use it to encode your media. For information on how to encode with **Media Encoder Premium Workflow** using **.NET**, see [Advanced encoding with Media Encoder Premium Workflow](../media-services-encode-with-premium-workflow).

##<a id="existing_workflows"></a>Modify existing workflows

You can start with default workflow files and update them using the designer tool. To get the files, go [here](https://github.com/Azure/azure-media-services-samples/tree/master/Encoding%20Presets/VoD).
