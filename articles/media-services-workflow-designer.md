#Workflow Designer

The **Workflow Designer** is a standalone tool that is used to create workflows/graphs for encoding with **Media Encoder Premium Workflow**.

You can also use this tool to modify existing workflows. 
The default workflow files can be found [here](https://github.com/Azure/azure-media-services-samples/tree/master/Encoding%20Presets/VoD).


>[AZURE.NOTE]To get a copy of this Workflow Designer tool, please contact mepd@microsoft.com.
