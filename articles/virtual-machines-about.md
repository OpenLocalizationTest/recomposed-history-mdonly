#About Azure Virtual Machines

Azure Virtual Machines is one of several types of on-demand, scalable computing resources offered by Azure. Typically, you'll choose a virtual machine instead of another option if you need more control over the computing environment than the other choices offer. To compare these choices, see [need link] 

Compared to using a virtual machine you host yourself, an Azure VM gives you the flexibility of virtualization without having to buy and maintain the hardware to run the virtual machine. However, you do need to maintain the virtual machine -- configuring, patching, and maintaining the operating system and any other software that runs on the virtual machine.
