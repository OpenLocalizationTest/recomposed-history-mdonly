# Deploy modules from Azure Automation from PowerShell Gallery

The [PowerShell Gallery](http://www.powershellgallery.com/) is the central repository for Windows PowerShell content.  You can find modules in the gallery that can be installed in your Azure Automation environment to be used by your runbooks.
