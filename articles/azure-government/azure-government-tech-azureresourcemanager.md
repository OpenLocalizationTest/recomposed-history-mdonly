#  Azure Resource Manager

The following information identifies the Azure Government boundary for Azure Resource Manager:

-------Insert table from Chris-----------

Azure Resource Manager (ARM) is generally available in Azure Government with the use of classic resources. The classic resources that are available in Azure Government are:
* Microsoft.ClasicCompute
* Microsoft.ClassicNetwork
* Microsoft.ClassicStorage
* Microsoft.Insights
* Microsoft.Keyvault
* Microsoft.Storage
* Microsoft.Authorization
* Microsoft.Features
* Microsoft.Resources
* Microsoft.Scheduler
* Microsoft.Web

For additional information, please see the <a href=https://azure.microsoft.com/en-us/documentation/articles/resource-manager-supported-services/#> Azure Resource Manager public documentation </a>.

<a href="https://blogs.msdn.microsoft.com/azuregov/">Microsoft Azure Government Blog. </a>
