# Supported services for Diagnostic Logs
The following services support Diagnostic Logs:

- Application Gateways
- Azure Search
- Data Lake Store
- Data Lake Analytics
- Key Vault
- Logic Apps
- Network Security Groups
- Software Load Balancer
