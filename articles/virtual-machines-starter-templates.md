# Starter templates

Get started using templates, quickly.

| Template Name | Description | View the template | Deploy it right now |
|:---|:---|:---:|:---:|
| Availability Set | Creates an Availability Set. | [link](https://github.com/Azure/azure-quickstart-templates/tree/master/101-create-availability-set) | <a href="https://deploy.azure.com/?repository=https://github.com/Azure/azure-quickstart-templates/tree/master/101-create-availability-set" target="_blank"><img src="http://azuredeploy.net/deploybutton.png"/></a> |
| Simple Ubuntu VM | Deploys a very simple VM running Ubuntu 14.10. | [link](https://github.com/Azure/azure-quickstart-templates/tree/master/101-simple-ubuntu-vm) | <a href="https://deploy.azure.com/?repository=https://github.com/Azure/azure-quickstart-templates/tree/master/101-simple-ubuntu-vm" target="_blank"><img src="http://azuredeploy.net/deploybutton.png"/></a> |
| Simple Windows VM | Deploys a very simple VM running Windows 2012 R2 Datacenter.| [link](https://github.com/Azure/azure-quickstart-templates/tree/master/101-simple-windows-vm) | <a href="https://deploy.azure.com/?repository=https://github.com/Azure/azure-quickstart-templates/tree/master/101-simple-windows-vm" target="_blank"><img src="http://azuredeploy.net/deploybutton.png"/></a> |
| VM with Empty Data Disk | Deploys a VM using an image that you specify and adds an empty data disk. | [link](https://github.com/Azure/azure-quickstart-templates/tree/master/101-vm-empty-data-disk-linux) | <a href="https://deploy.azure.com/?repository=https://github.com/Azure/azure-quickstart-templates/tree/master/101-vm-empty-data-disk-linux" target="_blank"><img src="http://azuredeploy.net/deploybutton.png"/></a> |
| VM from User Image | Creates a VM using your own custom image (must already be stored in a Storage Account). Will also deploy a Virtual Network, Public IP address, and a Network Interface. | [link](https://github.com/Azure/azure-quickstart-templates/tree/master/101-vm-from-user-image) | <a href="https://deploy.azure.com/?repository=https://github.com/Azure/azure-quickstart-templates/tree/master/101-vm-from-user-image" target="_blank"><img src="http://azuredeploy.net/deploybutton.png"/></a> |
