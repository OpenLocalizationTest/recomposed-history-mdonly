#Install and configure PowerShell for HDInsight

HDInsight PowerShell is integrated with Windows Azure PowerShell version 0.7.2. Although the updates are optional, they provides streamlined experience with using Windows Azure PowerShell. For example, the *Subscription* parameter is no longer needed.

Before installing Windows Azure PowerShell version 0.7.2 or higher, you must uninstall Windows Azure HDInsight PowerShell first. 

For information on installing and configuring Windows Azure PowerShell, see [How to install and configure Windows Azure PowerShell][powershell-how-to-install].
